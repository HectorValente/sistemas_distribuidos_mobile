COMANDO GIT

git init: Inicializa um novo repositório Git localmente.
git clone <URL>: Clona um repositório remoto para o seu computador.
git add <arquivo>: Adiciona um arquivo ao índice para ser rastreado pelo Git.
git commit -m "<mensagem>": Registra as mudanças feitas nos arquivos do repositório.
git status: Exibe o estado atual do repositório, incluindo arquivos modificados, adicionados ou removidos.heckout mainhe
git push: Envia as alterações locais para o repositório remoto.
git pull: Atualiza o repositório local com as alterações do repositório remoto.
git branch: Lista todas as branches no repositório local.
git checkout <branch>: Alterna para uma branch específica.
git merge <branch>: Combina as alterações de uma branch na branch atual.
git log: Exibe o histórico de commits do repositó
git reset --hard HEAD: Desfaz todas as mudanças locais e reverte para o último commit.
git remote -v: Lista os repositórios remotos configurados.
git fetch: Busca as alterações do repositório remoto, mas não as mescla com o repositório local.
git config: Configura o Git, permitindo definir variáveis como nome de usuário, email, etc.
git tag: Gerencia tags no repositório.
git diff: Mostra as diferenças entre arquivos.
git stash: Salva as mudanças locais temporariamente.
git rebase: Reaplica commits em uma branch base.
git rm: Remove arquivos do repositório.
git checkout -b <nome_branch>: Cria uma nova branch e muda para ela.
git revert <hash_do_commit>: Desfaz um commit específico, criando um novo commit que reverte as mudanças.
git remote add <nome_remoto> <URL_do_repositório>: Adiciona um novo repositório remoto ao seu projeto.
git branch -d <nome_da_branch>: Deleta a branch local
git pull origin <branch>: tras as atualizações da branch remota para a sua branch local de forma rápida e fácil.
git pull --tags origin master
